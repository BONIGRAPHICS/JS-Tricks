# ✨ JavaScript Tricks with HTML & CSS

Welcome to my collection of simple and useful JavaScript tricks!  
Each trick is created using **JavaScript**, **HTML**, and **CSS**, all in one file for ease of understanding.

This repo is perfect for:
- Beginners looking to learn JavaScript through visual examples
- Developers who love collecting small UI tricks
- Anyone who wants quick inspiration!

Happy learning! 🚀

---

## 📁 File Structure

Each file is named as `trickX.html`:
```
📁 js-tricks/
├── trick1.html
├── trick2.html
├── trick3.html
├── ...
```

All the code (HTML + CSS + JS) is inside each file.

---

## 🧪 How to Use

1. Clone the repo:
```bash
git clone https://github.com/your-username/js-tricks.git
```

2. Open any `trickX.html` file in your browser.
3. Explore and enjoy!
